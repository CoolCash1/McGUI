McGUI Setup - Help

Welcome to McGUI

Thanks for choosing McGUI, your all-in-one Minecraft Server Management dashboard that you can access on any device, whenever, wherever.

Note: Nothing is saved untill your done setup.

Password

Choose a password for your McGUI Instance. Make sure it is secure so only you can log in.

Server Address

Input the address of your server. McGUI will ping your server, so make sure its running.

Server Settings

Now that you've pinged your server, its time to add your server's directory (relative to McGUI).

One more thing

What would you like to name your server. Also add a system command to start your server from McGUI.

Done!

Your McGUI Instance is now restarting. Reload a couple times untill its done.

All done

McGUI is now ready for use. Click the button below to start using your server interface now.

Failed

McGUI setup failed. Please reload the page to restart. Your browser's dev tools might show more information in the Javascript console.
